# 🎧 Listen to Me

> *纸短情长，读给你听*

一款文档朗读应用，让文字化作声音，在耳畔低语。

---

## ✨ 功能特色

- 📄 **PDF / Word 文档解析** —— 上传即读，无需繁琐转换
- 🔊 **系统语音朗读** —— 调用浏览器原生 TTS，无需付费 API
- ⏯️ **播放控制** —— 播放、暂停、停止、下一篇，随心掌控
- 🔄 **列表循环** —— 一遍又一遍，直到入眠
- ⏰ **定时停止** —— 5~90 分钟，伴你安眠
- 🎛️ **声音调节** —— 音色、语速、音高、音量，打造专属听感

---

## 🌙 使用场景

| 场景 | 描述 |
|------|------|
| 📚 学习 | 听论文、听报告、听资料，解放双眼 |
| 🌙 睡前 | 听小说、听散文，定时关闭，安然入梦 |
| 🚶 通勤 | 听文档、听笔记，碎片时间高效利用 |
| 👴 无障碍 | 帮助视障人士阅读文字内容 |
